## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

 - Add transactions
 - List transactions (history)
 - Delete transactions
 - Balance
